# Social-Media-Networking-Database
Relational Database project for social media networking 

**Social Media Networking Database (SQL Project)**

📌 Project Overview

This project is a Social Media Networking Database designed using SQL.
It is Similar to like a real-world social media platform where users can create profiles, post content, follow others, and interact through likes and comments.

---------------------------------------------------------------------------------------------------------------------------------

🎯 Objectives

- Design a relational database for social media system
- Manage users, posts, comments, and relationships
- Perform data analysis using SQL queries

---------------------------------------------------------------------------------------------------------------------------------

🛠️ Tech Stack

- SQL (Oracle / SQL Server)
- DBMS Concepts (Relationships, Constraints, Joins, Subqueries)

------------------------------------------------------------------------------------------------------------------------------------

🗂️ Database Structure

The database includes following tables:

- Users
- Posts
- Comments
- Likes
- Followers

----------------------------------------------------------------------------------------------------------------------------------

🔗 Key Features

- User profile management
- Post creation and engagement
- Followers & following system
- Query-based data analysis

------------------------------------------------------------------------------------------------------------------------------------

📊 Sample Queries

- Retrieve all posts of a user
- Find most active users
- Get followers count
- Join queries for user interactions

-------------------------------------------------------------------------------------------------------------------------------------

▶️ How to Run

1. Open SQL Developer Server (Oracle )
2. Import the ".sql" file
3. Execute queries step by step

-----------------------------------------------------------------------------------------------------------------------------------

💡 Learning Outcomes

- Improved SQL query writing skills
- Understanding of database design
- Hands-on experience with joins and relationships

-------------------------------------------------------------------------------------------------------------------------------------

📌 Conclusion

This project demonstrates how a real-world social media database is structured and managed using SQL.
